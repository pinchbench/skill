# PinchBench Benchmark Skill

PinchBench measures how well different LLM models perform as the "brain" of an OpenClaw agent. Like SWE-Bench or TerminalBench, it answers: _"How good is Model X when used as the default model for OpenClaw?"_

Results are collected on a central server at api.pinchbench.com with a public leaderboard on pinchbench.com.
